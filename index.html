Proyectos

Sitio Web para NM Nails Salón

Un sitio web diseñado para un salón de belleza que tiene como propósito ofrecer información sobre los servicios, horarios y ubicación del establecimiento, así como permitir a los usuarios solicitar citas en línea.

Auto Drip

Aplicación web que se integra con un sistema de riego automatizado,que además monitorear en tiempo real la temperatura y la humedad del suelo. La aplicación permite visualizar datos, gestionar el riego y almacenar información en una base de datos.

POS El Moro

Sistema de punto de venta (POS) personalizado para el restaurante El Moro, este sistema incluye características como inicio de sesión seguro, gestión de usuarios con diferentes niveles de acceso, y la capacidad de tomar pedidos desde tablets

Sitio Web Anúnciate & Busca

Es una plataforma de directorio de comercios que permite registrar negocios con información detallada y catálogos de productos. Incluye búsqueda avanzada, inicio de sesión para usuarios y un sistema de reseñas y calificaciones.

Asistente Virtual Educativo con IA

Es una herramienta de inteligencia artificial que mejora el aprendizaje. Con una interfaz web accesible desde móviles y computadoras, ofrece funciones como corrección ortográfica, traducción, generación de imágenes y conversión de texto a audio.
